# 🤖 Arduino Projects Collection

Welcome to my Arduino projects repository. This repository contains Arduino-based projects developed and simulated using **Tinkercad**. Each project demonstrates the application of sensors, actuators, and embedded programming to solve practical problems.

---

## 📚 Projects Included

### 🔥 1. Fire Fighting Robot

An autonomous robot that combines **obstacle avoidance** and **fire detection**. The robot detects obstacles using an ultrasonic sensor and identifies fire using a flame sensor. When fire is detected, a servo motor simulates a fire-extinguishing mechanism while an LCD provides real-time system status.

**Key Features**
- Obstacle avoidance
- Fire detection
- Servo-controlled fire extinguisher simulation
- LCD status display
- Autonomous navigation

**Technologies**
- Arduino Uno
- HC-SR04 Ultrasonic Sensor
- Flame Sensor
- Servo Motor
- 16×2 LCD
- DC Motors
- Motor Driver

**Simulation**

### 📏 2. Ultrasonic Distance Sensor with Servo

An Arduino-based object detection system that measures distance using an HC-SR04 ultrasonic sensor. According to the detected distance, LEDs indicate proximity while a servo motor rotates to predefined angles.

**Key Features**
- Real-time distance measurement
- LED distance indication
- Servo motor control
- Serial Monitor output

**Technologies**
- Arduino Uno
- HC-SR04 Ultrasonic Sensor
- Servo Motor
- LEDs
- Breadboard

**Simulation**
https://www.tinkercad.com/things/8KWKKqeZBV0-ultrasonic-distance-sensorservo
